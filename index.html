PASOS PARA EJECUTAR EL BOT EN TU PC
1

Crear bot en Telegram

Buscá @BotFather → escribí /newbot → elegí nombre → copiá el token.

2

Obtener tu Chat ID

Buscá @userinfobot → mandá cualquier mensaje → copiá tu Chat ID.

3

Instalar Python

Descargá desde python.org. En Windows marcá "Add to PATH" durante la instalacion.

4

Instalar librerias

Abrí la terminal (cmd en Windows) y ejecutá:

pip install yfinance pandas requests schedule
5

Copiar el codigo del bot

Copiá el codigo de abajo → pegalo en balanzbot.py → completá tu token y chat ID.

6

Ejecutar

En la terminal: python balanzbot.py — Las alertas llegan solas a Telegram.
